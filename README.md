# flutter_dmcb_logger
[![License](https://img.shields.io/github/license/mashape/apistatus.svg)](https://github.com/zenganiu/flutter_dmcb_logger)
## **Flutter日志库**




## 特性
* IDEA使用`Grep Console`插件支持内容折叠。
* App支持查看日志记录。

## 安装
在 `pubspec.yaml` 中添加
```yaml
dependencies:
  # 最新版本
  flutter_gps:
    git: https://github.com/zenganiu/flutter_dmcb_logger.git
```
